# Books - Simple REST API.

## List of functionalities:
- Get all books.
- Create and add book.
- Update book.
- Delete book.

# Routes
| Route | Functionality |
| --- | ---- |
| /books/ (GET) | Get all books. |
| /books/ (POST) | Create and post a book. |
| /books/:id (GET) | Get book by id. |
| /books/:id (PUT) | Update book by id. |
| /books/:id (DELETE) | Delete book by id. |
